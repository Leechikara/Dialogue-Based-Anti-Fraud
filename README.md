# Dialogue-Based-Anti-Fraud
Code and data for EMNLP-IJCNLP 2019 paper "Are You for Real? Detecting Identity Fraud via Dialogue Interactions".

### Requirements
     python 3.6
     pytorch >= 0.4.1
     matplotlib
     numpy

### Training model
     cd src
     ./rl.sh (or ./warm_up.sh)

### Test model
     ./test.sh
